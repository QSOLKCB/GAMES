TERNARY DRIFT - NATIVE VERTICAL SLICE

Run TERNARY.EXE on 64-bit Windows 7 or later. No installation, network,
browser, or external game runtime is required.

FLIGHT
  W/S          thrust / reverse
  A/D          lateral strafe
  Left/Right   rotate
  Space        fire pulse cannon
  Shift        cruise thrust
  K            toggle assisted flight / engine kill
  E            dock or tractor nearby salvage
  J            use a nearby jump gate

DOCKED
  Up/Down      select commodity
  B / S        buy / sell one unit
  M            accept a generated contract
  U            buy the displayed upgrade
  R            repair hull and recharge shield
  L            launch

FILES
  F5 / F6      save / load TERNARY.SAV
  F9           begin a fresh deterministic recording; press again to stop
  F10          replay TERNARY.RPL and verify its final state hash

Optional seed: TERNARY.EXE 0x5445524E41525931

This is an early, genuine C99/Win32 implementation of the larger design.
See the source README for implemented systems and explicit deferrals.
